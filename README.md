# getnadir.dev

Landing page for [NadirClaw](https://github.com/doramirdor/NadirClaw), an open-source LLM router with built-in observability.

## Deploy

Static site. No build step required.

- **Netlify**: Connect this repo, publish directory: `/`, done.
- **Local**: `python3 -m http.server 8765`

## Features

- Dynamic GitHub star count (fetched from GitHub API)
- Single `index.html` with inline CSS
- Mobile responsive
- No dependencies
